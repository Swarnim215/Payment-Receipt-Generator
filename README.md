# Payment-Receipt-Generator

A simple Python project that generates payment receipts using user input.
The receipt includes date & time using the `datetime` module and is saved
as a text file.

## Features
- Generates payment receipt in text format
- Uses `datetime` for real-time date & time (and not the time module as datetime module is high level and often useful for Current date & time; Formatting dates; Comparing dates; Invoices, receipts, logs)
- Saves receipt AUTOMATICALLY with receipt number
- Beginner-friendly Python project

## Technologies Used
- Python 3
- datetime module

## How to Run
```bash
python receipt_generator.py
